# Contributing to The Serial Bioinformatics

Thank you for your interest in contributing to this book! We welcome corrections, improvements, and new content.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a branch** for your changes (`git checkout -b my-new-feature`).
4.  **Make your changes** to the Markdown files.
5.  **Preview your changes** locally (see below).
6.  **Commit your changes** (`git commit -am 'Add some feature'`).
7.  **Push to the branch** (`git push origin my-new-feature`).
8.  **Create a Pull Request**.

## Running the Book Locally

To ensure your changes look correct, please build the book locally before submitting a Pull Request.

1.  Install the requirements:
    ```bash
    pip3 install -r requirements.txt
    ```
2.  Run the development server:
    ```bash
    python3 -m mkdocs serve
    ```
3.  Open `http://127.0.0.1:8000` in your browser.
